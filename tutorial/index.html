Activator UI 利用の注意点

Code・Tutorial 以外は利用しないでください。
Activator UI で Lagom がサポートされていないため、おかしな挙動を引き起こします。

  1. Code を開いて下さい

    このチュートリアルが閉じてしまった場合は、右上のメニューから表示してください。

  2. チュートリアルをいっぱいまで広げてください

  3. 「Lagom ハンズオン」を上部メニューから開いて下さい

ハンズオン資料の使い方

スライドの操作

上記のチェックが入っていると講演者のスライドと表示が同期します。 キーボードの ←・→ で、スライドを戻したり進めたりすることができます。

スライドの操作をすると、同期が解除されます。 再び表示を同期する場合はチェックボックスをONにしてください。

質問したいとき

急ぎの場合は、手を上げてお近くのスタッフまでお声がけください。

こちらのフォームに質問を入力頂くと、ハンズオン終了後に講師から質問の回答をさせて頂きます。 既に投稿されている質問に共感するものがあればをお願いします。
※ こちらのフォームへの質問は匿名になります。
